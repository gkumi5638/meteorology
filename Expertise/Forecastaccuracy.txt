予報精度評価法
・カテゴリー予報

  予想 現象有   実況 現象有 ＝ A
  予想 現象なし 実況 現象有 ＝ B
  予想 現象有   実況 現象なし ＝ C
  予想 現象なし 実況 現象なし ＝ D

  的中率 (A + D) / (A + B + C + D)
  空振り率 C / (A + B + C + D)
  見逃し率 B / (A + B + C + D)
  スレットスコア A / (A + B + C)
  バイアススコア (A + C) / (A + B)

・注意報・警報予報の予報精度

  予想 現象有   実況 現象有 ＝ A
  予想 現象なし 実況 現象有 ＝ B
  予想 現象有   実況 現象なし ＝ C
  予想 現象なし 実況 現象なし ＝ D

  予想現象有の合計  A + C = E
  予想現象有の合計  B + D = F
  実況現象有の合計  A + C = E
  実況現象有の合計  B + D = F

  的中率(注意報予報) A / E
  空振り率(注意報予報) C / E
  見逃し率(注意報予報)

  平均誤差
  2乗平均平方根誤差(RMSE)
・降水確率予報と実況での降水有無に関する予報精度評価
